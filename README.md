# Trip Palette

行ったことのある場所と行きたい場所を、カテゴリ・周辺マップ・観光スポット・旅のTODOでまとめる静的Webサイトです。

## 使い方

```bash
python3 -m http.server 4173
```

ブラウザで `http://localhost:4173` を開いて確認します。

## 主な機能

- 行きたい場所のカテゴリ別フィルター
- 選択スポットに連動する周辺マップ表示
- 周辺観光スポット、季節、優先度、予算メモの表示
- 行った場所のタイムライン記録
- 行った先でやりたいことを追加できるTODOリスト
- 予算、持ち物、同行者、優先度などの拡張アイデア表示
